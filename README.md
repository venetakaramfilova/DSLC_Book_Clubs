# Data Science Learning Community Book Clubs

## Data Science Learning Community Book Club
A friendly weekly meet, organized by the [Data Science Learning Community](https://dslc.io/bookclubs.html) where participant read a single book, covering a data-related topic.

## Objective
Keep a diary of the books read, code and exercises tested.

## Structure
Folders are organized chronologically by book read, and contain a visual and the code, used for creating it.
